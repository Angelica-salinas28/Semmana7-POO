# Sistema de Tickets – Uso de Constructores y Destructores en Python

## Descripción del programa
Este proyecto implementa un **sistema sencillo de tickets** que permite registrar la información de clientes y guardar los datos en un archivo de texto.  
El objetivo principal es **demostrar el uso de constructores (`__init__`) y destructores (`__del__`) en Python**, para lo cual aplicando la arquitectura **Modelos / Servicios** vista en clase.

El programa muestra dos escenarios:
1. Cuando el programador **olvida cerrar un recurso** (archivo).
2. Cuando el recurso se **cierra correctamente de forma explícita**.
